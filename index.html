This portfolio documents my projects and learnings from the "Generative AI for Work & Research" course, showcasing my ability to apply no-code AI tools for task automation and research productivity.

About the Course

An 8-week deep dive into using no-code AI tools for task automation and enhanced productivity in work and research.

Module 1: Websites, Documents & Presentations

Understanding LLMs to generate text, create no-code websites, and design presentations.

Module 2: AI-Automated Data Analysis

Using AI tools to analyze data, manipulate files, and create automated reports and dashboards.

Module 3: GenAI for Research & Text Analysis

Applying AI for literature searches, text analysis, and knowledge synthesis from large document collections.

Module 4: Custom GPTs and Chatbots

Building custom chatbots and research assistants using advanced LLM architectures.

Module 5: AI for Task Automation

Automating repetitive tasks through AI-driven flows and agent-based systems.

Module 6: GenAI for Communication and Outreach

Using AI to create research summaries, scientific figures, and video abstracts for dissemination.

Module 7: Capstone Project & Showcase

Developing and presenting a final AI solution or tool, integrating all learned concepts in a comprehensive project.
